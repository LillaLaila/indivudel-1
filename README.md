# indivudel-1


# Environment

- **Python version:** 3.13.7

- **Required packages:** pandas, matplotlib, numpy, scipy (listed in `requirements.txt`)



## Getting Started

```bash
# Clone the project
git clone <link>
cd <project folder name>

# Create and activate a virtual environment
python -m venv .venv
# Windows PowerShell
.venv\Scripts\Activate

# Install dependencies
python -m pip install -r requirements.txt
```
